# questboard

A new Flutter project. Meant to help with productivity and gamification.  Intended for educational purposes and to challenge myself to learn more about app development. 

## Getting Started

This project is not a starting point for a Flutter application.

To run this project, you need to have the apk installed on your android device.

Find the apk in the releases section of the codebase, under the build/app/outputs/flutter-apk/ directory. Download it and install it on your device.
